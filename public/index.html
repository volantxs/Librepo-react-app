Features

RXP

Reader's will gain RXP points with each book added to completed collection. Initially the RXP will be accumulated as single type which would disintegrate into multiple categories in future with respect to the theme of the book.

Levels

Readers will set their monthly/yearly goals such that each completion of the goal will allow you to climb a new level defined by the system. We would like to make it look like a pseudo-game web app as much as possible in coherence with the users experience

Stats

In later phases, the plan is to provide monthly stats to the users to keep a track of their reading pattern.
